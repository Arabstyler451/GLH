# Greenfield Local Hub Web App

ASP.NET Core MVC Web Application using C# and .NET 9.0.

---

## Requirements

Before running the project, ensure the following are installed:

- Visual Studio 2022 (version 17.12 or later)
- .NET 9.0 SDK
- ASP.NET and Web Development workload in Visual Studio
- SQL Server LocalDB

----------------------------------------------------------------------------------------------

## Required NuGet Packages

Ensure the following NuGet packages are installed:

- Microsoft.AspNetCore.Authentication.Google (9.0.12)
- Microsoft.AspNetCore.Diagnostics.EntityFrameworkCore (9.0.0)
- Microsoft.AspNetCore.Identity.EntityFrameworkCore (9.0.12)
- Microsoft.AspNetCore.Identity.UI (9.0.12)
- Microsoft.EntityFrameworkCore.Sqlite (9.0.12)
- Microsoft.EntityFrameworkCore.SqlServer (9.0.12)
- Microsoft.EntityFrameworkCore.Tools (9.0.12)
- Microsoft.VisualStudio.Web.CodeGeneration.Design (9.0.12)
- Microsoft.EntityFrameworkCore.Design (9.0.12)

----------------------------------------------------------------------------------------------

## Required Actions

In the NuGet Package Manager Add a migration and update database:

e.g. First do this -> Add-Migration testing   Then do this -> Update-Database

----------------------------------------------------------------------------------------------

Lastly run the project with http not https on the toolbar at the top of Visual Studio
